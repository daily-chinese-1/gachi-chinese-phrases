# ガチ中国語単語500 - プロジェクトガイド

## 概要
ネイティブが本当に使う中国語の実戦単語500個を収録したWebベースの単語帳ツール。
スマホ特化のUI、ジャンル別フィルタ、検索機能、ネイティブ音声再生を搭載。

## コンセプト
- 「実戦」特化型: 書籍の単語帳にない、ネイティブが日常で本当に使う表現を厳選
- 場面（シーン）別でサッと引ける、ストレスフリーなデジタル単語帳
- ワンタップで音声再生、即時アクセス

## ファイル構成
```
index.html          # メインアプリ（HTML/CSS/JS一体型、単一ファイル）
data.csv            # 単語データ（500語）
audio/              # 音声ファイル（m4a、1000ファイル）
  ├── 1単語.m4a     # 各番号の単語音声
  ├── 1例文.m4a     # 各番号の例文音声
  ├── ...
  └── 500例文.m4a
images/             # パンダキャラクター画像
  ├── panda-teach.jpg  # ヘッダーアイコン（指示棒パンダ）
  ├── panda-cry.png    # 検索0件時（泣きパンダ）
  └── panda-good.png   # 予備（ウインクパンダ）
```

## data.csv のフォーマット
```
番号,ジャンル,単語,拼音,日本語訳,補足説明,例文,例文の修正案
```
- ジャンル: あいづち・リアクション / 日常会話・雑談 / 感情表現 / 人間関係 / SNS・ネットスラング / 食事・飲み会 / 旅行・買い物
- 例文は「中国語 / 日本語訳」の形式（スラッシュ区切り）
- カンマを含むフィールドはダブルクォートで囲む
- 最終行にも改行を入れること（欠けると最後の行が読み込まれない）

## 音声ファイルの命名規則
- `{番号}単語.m4a` - 単語の発音
- `{番号}例文.m4a` - 例文の発音
- 番号はdata.csvの1列目の番号と一致させる

## 修正時の注意点
- index.htmlは単一ファイル構成。CSS・JS・HTMLが全て含まれる
- CSVの読み込みにはキャッシュバスティング(`?v=`+タイムスタンプ)を使用
- 音声は`playAudio(番号, '単語'or'例文', ボタン要素)`で再生制御
- ジャンルの色はCSS内の`.genre-*`クラスで管理
